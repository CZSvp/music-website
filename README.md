# Mi Sitio Web Musical 🎵

Bienvenido a tu página web personal para promocionar tu música y redes sociales.

## 📋 Contenido

- **index.html** - Estructura principal de la página
- **styles.css** - Estilos y diseño responsive
- **script.js** - Funcionalidades interactivas

## 🎨 Características

✅ Diseño moderno y responsive  
✅ Navegación suave  
✅ Sección de galería de canciones  
✅ Enlaces a redes sociales  
✅ Formulario de contacto  
✅ Animaciones personalizadas  

## 🚀 Cómo usar

1. Clona este repositorio
2. Abre `index.html` en tu navegador
3. Personaliza el contenido según tus necesidades

## 🔧 Personalización

### Cambiar colores
Edita las variables en `styles.css`:
```css
:root {
    --primary-color: #6c5ce7;
    --secondary-color: #00b894;
    --dark-color: #2d3436;
    --light-color: #f5f6fa;
    --text-color: #2d3436;
}
```

### Actualizar redes sociales
Reemplaza las URLs en la sección de redes sociales en `index.html`

### Agregar canciones
Duplica las tarjetas de música y actualiza:
- Nombre de la canción
- Descripción
- Enlaces a plataformas de streaming

## 📝 Notas

- Los enlaces del formulario de contacto actualmente muestran un alert
- Puedes integrar un servicio de email para funcionalidad completa
- Personaliza el favicon y título según tu marca

¡Disfruta promocionando tu música! 🎶
